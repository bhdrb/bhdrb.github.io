# PIXEL ONE 홈페이지 디자인 아이디어

## 선택된 디자인: 미니멀리스트 모던 (Minimalist Modern)

### 디자인 철학
**PIXEL ONE**은 단순함과 정확성을 추구하는 기업이므로, 미니멀리스트 모던 디자인을 선택했습니다. 이 접근법은:
- 한 개의 픽셀이 하나의 완벽한 점으로 표현되는 개념을 시각적으로 강화
- 불필요한 요소를 제거하고 핵심만 남기는 철학
- 깨끗하고 명확한 시각 계층 구조

### 핵심 원칙
1. **극도의 단순성**: 모든 요소가 목적을 가짐
2. **정밀한 정렬**: 그리드 기반의 완벽한 배치
3. **공간의 힘**: 여백을 적극 활용하여 호흡감 제공
4. **흑백 기조**: 명확성과 전문성을 표현

### 색상 철학
- **배경**: 순수한 흰색 (rgb(255, 255, 255))
- **텍스트**: 깊은 검은색 (rgb(20, 20, 20))
- **악센트**: 차분한 회색 (rgb(100, 100, 100))
- **픽셀 로고**: 선명한 검은색 단일 점

### 레이아웃 패러다임
- 중앙 정렬 구조로 시각적 안정감 제공
- 상단: 픽셀 로고 + 회사명
- 중단: 여백 (호흡감)
- 하단: 개인정보처리방침 링크

### 시그니처 요소
1. **픽셀 로고**: 정사각형 프레임 내 단일 검은 점
2. **타이포그래피**: 세리프 폰트로 전문성 강조
3. **하이라인**: 링크 위의 미세한 언더라인 애니메이션

### 상호작용 철학
- 마우스 호버 시 링크 색상 변화
- 팝업 오버레이는 반투명 검은색 배경
- 부드러운 페이드 인/아웃 애니메이션

### 애니메이션
- 페이지 로드: 페이드 인 (0.6초)
- 링크 호버: 색상 전환 (0.3초)
- 팝업 오픈: 스케일 + 페이드 (0.4초)

### 타이포그래피 시스템
- **제목**: Playfair Display (세리프, 고급스러움)
- **본문**: Inter (산세리프, 가독성)
- **링크**: Inter 일반체, 언더라인 호버 효과
